# Findings - Vaultify

## Research & Discoveries
- Project detected as a Vite + TypeScript + Tailwind CSS application.
- **Requirement 1:** Centralized family document vault.
- **Requirement 2:** Integrated with Google Drive.
- **Requirement 3:** Data stored/indexed in a JSON file hosted on GitHub Pages.
- **Requirement 4:** Deployment via GitHub Actions to `gh-pages`.
- **Requirement 5:** UX: Quick, snappy, previews enabled.

## Constraints
- Needs to handle Google Drive authentication or public/service-account access.
- GitHub Pages is static; search/filter logic must be client-side.
- Previews must be handled via GDrive's webViewLink or similar embedding.
